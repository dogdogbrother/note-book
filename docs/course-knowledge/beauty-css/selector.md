## 层次选择器

* 相邻选择器 - `elem1+elem2`,`elem2`只有紧贴着在`elem1`后面才会被生效.
* 通用选择器 - `elem1~elem2`,`elem2`只要是在`elem1`后面就会生效.

## 集合选择器

* 并集选择器 - `elem1,elem2`,`elem1`和`elem2`两个元素都会被选择到.
* 交集选择器 - `elem.class`,只这个元素同时拥有此2种选择器.

## 状态选择器

* `:active` - 鼠标激活的元素,也就是鼠标按下和抬起之间.
* `:link` - 未访问的链接元素.
* `:visited` - 已访问的链接元素

## 结构选择器
* `:empty` - 没有子元素的元素
